July 19, 2020 字数 167

每次更新hugo博客所需步骤

先cmd到本地博客项目的文件夹

cd C:\Users\mapleTan\programming\mapleBlog\hugo\bin\test3

新建一个md文件

hugo new posts/FileName.md

写完博文后在博客根目录下更新搜索索引

npm run index

输入命令把博客build起来

hugo

本地查看更新后的博客

hugo server -t Diaspora --buildDrafts

接着进入public文件夹

cd public

输入git指令

git add .
git commit -m "yyyy/mm/dd-hh:mm"
git push

完事
